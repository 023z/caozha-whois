# 草札域名Whois查询类（PHP）

草札域名Whois查询类（PHP），支持全球大部分域名的whois查询，支持中文域名在内的多种域名后缀，包括：.com，.net，.cn，.com.cn，.org，.biz，.cc，.mobi，.中国，.公司等等几百种域名后缀。
